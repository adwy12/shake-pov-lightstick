# Publication Boundary

This public repository intentionally includes only materials that are useful for reproducing the hand-waved POV lightstick prototype:

- Arduino firmware written for the prototype.
- Web Serial tuning UI.
- Hardware and protocol notes that describe the prototype behavior.

It intentionally excludes:

- APK files.
- Raw Bluetooth or RF captures.
- Proprietary firmware images.
- Decompiled commercial app source.
- Credentials, tokens, serial numbers, or private device identifiers.
- Research notes for unrelated lightsticks.

The goal is to present the project as an inspectable open hardware/software experiment while keeping third-party materials and private reverse-engineering artifacts out of the public repository.
